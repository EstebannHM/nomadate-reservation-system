# Nomadate Reservation System

A full-stack web application built with **ASP.NET Core** and **SQL Server** for managing hotel reservations, room availability, and user profiles.

## Features
- Room booking and availability management
- Client registration and authentication
- Reservation history and admin panel
- SQL Server integration for data persistence

## Technologies
- C# / ASP.NET Core MVC
- SQL Server
- HTML, CSS, JavaScript
