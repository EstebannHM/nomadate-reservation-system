# Imágenes Necesarias para el Sitio

Coloca las siguientes imágenes en esta carpeta (`wwwroot/images/`):

## Página de Inicio (Index)

### Imagen Destacada Principal:
- **hotel-featured.jpg** - Imagen grande del hotel (1200x500px recomendado)
  - Esta es la imagen principal que va debajo de la barra de búsqueda

### Cards de "Por Qué Elegirnos" (4 imágenes):
- **room-luxury.jpg** - Habitación de lujo
- **room-comfort.jpg** - Imagen que muestre comodidad (cama, almohadas)
- **room-service.jpg** - Servicio de habitación o recepción
- **room-location.jpg** - Vista exterior o ubicación del hotel

## Tamaños Recomendados:
- Logo: Ya tienes `logo.jpeg` ✓
- Imagen destacada: 1200x500px (ancho x alto)
- Cards de características: 400x300px cada una

## Formato:
- JPG o PNG
- Calidad alta pero optimizado para web

## Nota:
Mientras agregás las imágenes, el sitio mostrará espacios en blanco o imágenes rotas, pero el diseño y funcionalidad están completos.
